# Storage : Permission
## Command
```
chmod [option] [file or dir]
```
## Permission 
| permission | action | option (letter) | option (number)|
|:---:|:---:|:---:|:---:|
| read | view or open | r | 4 |
| write | edit | w  | 2 |
| execute | execute or run | x | 1 |
