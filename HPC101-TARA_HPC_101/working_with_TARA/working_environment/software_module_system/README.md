# Software : Module system
## Useful commands
| command | description |
|--:|:--|
|`module avail`| list all available modules|
|`module load [module]` | load `[module]`|
|`module list` | list all loaded module(s) |
|`module unload [module]` | unload `[module]` |
|`module purge` | unload all module(s) |
