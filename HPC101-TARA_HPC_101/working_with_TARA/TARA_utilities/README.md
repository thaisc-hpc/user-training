# TARA Utilities
## sinfo
To check available computing resources
```
$ sinfo
```

## squeue
To see job status in the queueing systems
```
$ squeue
```

## sacct
To check job status
```
$ sacct
```

## sbalance
To see check remaining balance
```
$ sbalance
```
or using `-d` to see detail in the user level. 
```
$ sbalance -d
```
  
## myquota
To check available space
```
$ myquota
```
